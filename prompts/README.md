# 代理提示词模板

## 目录

| 类型 | 文件 | 用途 |
|------|------|------|
| 调查者 | `investigator.md` | 独立探索、信息收集 |
| 评审员 | `reviewer.md` | 质量评审、安全验证 |
| 修复者 | `fixer.md` | 问题修复、代码改进 |
| 报告者 | `reporter.md` | 结果汇总、报告生成 |

## 使用原则

1. **模板是起点，不是终点** — 根据具体任务调整
2. **Schema 指令必须保留** — `必须用 JSON 回复: {...}` 是契约
3. **上下文要具体** — 模板中的 `{placeholder}` 必须替换
4. **语气要一致** — 整个 Workflow 使用相同的语气和格式
